# Design System
---

<img src={atomicDesign} alt="Atomic Design" />

- **Tokens**: The term "Token" in frontend development, particularly in the context of Design Systems, 
refers to variables that define the visual elements of a design such  as colors, spacing, 
fonts, shadows, etc. Tokens play a crucial role in connecting the entire process, from design 
to product development. This improve consistency, implementation, efficiency and scalability.
