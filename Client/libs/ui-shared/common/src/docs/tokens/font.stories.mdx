# Fonts
Our font system simplifies typography by utilizing a base font size of body: 16px. 
This approach enables responsive design as the font sizes for other elements in the app 
(h1, h2, h3, p, lead, etc.) are calculated using relative sizes, specifically using rem values. 
Additionally, our system calculates the line height to ensure optimal readability and proper 
spacing between lines, supporting WCAG guidelines for accessibility efforts.
