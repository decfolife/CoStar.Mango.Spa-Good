# Utility Classes
---

In web development, CSS utility classes play a crucial role in streamlining the styling process 
of your web development. These classes offer a set of pre-defined styles that can be easily 
applied to HTML elements, eliminating the need for writing custom CSS for every styling detail. 
This approach not only saves time but also promotes consistency across the project by 
adhering to a standardized set of styles. Just like other instances, these utility classes 
are based on design tokens.
