# Multiple Properties
---

The mixin for **multiple properties is `costar-alias()`** and builds on top of the `costar()` mixin. 
It facilitates quickly building of multiple CSS properties for your custom components.

The mixin parameter comes from the SCSS maps available at 
 `./common/src/styles/abstract/_aliases.scss`.
