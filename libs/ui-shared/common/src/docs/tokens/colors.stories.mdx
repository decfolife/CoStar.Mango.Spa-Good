# Color Palette
Our color palette complies with the Web Content Accessibility Guidelines (WCAG) standards, 
ensuring improved readability with suitable contrast ratios.

For new color combinations, please ensure that they meet one of the following criteria:

- [WCAG 2.0 AA contrast ratio](https://www.w3.org/TR/WCAG20/#visual-audio-contrast-contrast), or
- [WCAG 2.0 AAA contrast ratio](https://www.w3.org/TR/WCAG20/#visual-audio-contrast7)
